# 自爆プラグイン
# 概要
プレイヤーがモブを攻撃したとき罰として自爆させます。
# 利用したソフト
 IntelliJ
# 動画
こちらからツイッターに飛び再生してください。https://twitter.com/i/status/1389463251377741825
# 使い方
Spigot サーバーにプラグインを入れた時点で動作します。
/donta set power [威力] で爆発の威力を設定できます。
